JavaScript (JS)

an object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

JavaScript operators are used to assign values, compare values, perform arithmetic operations, and more. ex: +, -, %, +=

Variables

JavaScript variables are containers for storing data values. In this example, x is a variable:

var x = 5

Variable Naming Conventions

use common sense camelCase for identifier names (variables and functions). ex: variableName, camelCase.

Functions

A JavaScript function is a block of code designed to perform a particular task. A JavaScript function is executed when "something" invokes it (calls it). A JavaScript function is defined with the function keyword, followed by a name, followed by parentheses (). Function names can contain letters, digits, underscores, and dollar signs (same rules as variables). The parentheses may include parameter names separated by commas:(parameter1, parameter2, ...) The code to be executed, by the function, is placed inside curly brackets: {}. ex:

function myFunction (p1, p2) {return p1 * p2;}

Methods

JavaScript methods are actions that can be performed on objects. A JavaScript method is a property containing a function definition. You access an object method with the following syntax:

objectName.methodName()

Parameters

Function parameters are the names listed in the function definition. In this example, number1 and number2 are parameters:

function myFunction(number1, number2) {code to be executed};

myFunction(3, 5);

Arguments

Function arguments are the real values passed to (and received by) the function. In this example, 3 and 5 are the argument:

function myFunction(number1, number2) {code to be executed};

myFunction(3, 5);

Return

The return statement stops the execution of a function and returns a value from that function. ex:

function myFunction() {

return value;}
